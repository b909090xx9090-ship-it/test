왜 날개인가요?
건설·제조·물류 현장의 일용직 매칭을 가장 빠르고 정확하게
⚡
즉시 매칭 시스템
AI 기반 자동 매칭으로 공고 등록 즉시 적합한 인재를 추천합니다. 기술·경력·거리를 종합 분석한 점수 기반 매칭.
🏬
전국 대행소 네트워크
지역 거점 대행소를 통해 현장 인근 인재를 즉시 파견합니다. 크로스 매칭으로 타 지역 전문 인재도 조달 가능.
🛡️
검증된 인재 데이터베이스
자격증·경력을 대행소가 직접 검증한 신뢰할 수 있는 인재풀. 5단계 등급 시스템으로 우수 인재를 식별.
🧾
세금 자동 계산
일용직 원천징수, 부가세, 법인세를 자동 계산. 국세청 홈택스 연동 전자세금계산서 발행까지 원스톱 처리.
💰
투명한 정산 관리
일당·소개비·크로스 수수료를 자동 추적. 미수금 현황을 실시간 모니터링하고 정산을 체계적으로 관리.
📍
위치 기반 검색
현장 위치에서 반경 N km 이내 공고만 필터링. GPS 기반 최단거리 현장을 자동으로 추천합니다.

유형등록 식별자역할 및 특이사항
🏢 본사 사업자등록번호 플랫폼 최상위 운영 주체
🏬 대행소 사업자등록번호 구인자·구직자를 관리하며 구직자에게 소개비 수취. 대행소코드 발급
👔 구인자 사업자등록번호 대행소코드 입력으로 소속 설정
👷 구직자 주민등록번호 자격·경력·특기를 증빙자료와 함께 가입. 대행소코드 입력으로 소속 설정
🔑 로그인 방법: 아이디 · 이메일 · 소셜 등 다양한 방법 지원
  • 구인자·구직자는 대행소코드를 입력하여 하나의 대행소에 소속된다.
  • 구직자의 대행소코드는 본사 또는 해당 대행소가 언제든 변경 가능하다.
  • 부수 정보 입력 항목: 핸드폰번호, 이메일, 주소, 전화번호, 위경도, 닉네임, 계좌번호, 마지막 IP 등
  • 대행소·구인자·구직자는 로그인 차단 또는 삭제 처리 가능
  • 삭제는 레코드를 직접 삭제하지 않고 컬럼에 표기만 함 (Soft Delete)
구직자 등급 평가
  • 평가 주체: 구인자 또는 대행소
  • 등급: 5단계 / 대행소가 승급 또는 강등
  • 등급은 매칭·노출·수수료 등 시스템에 영향을 주지 않으며 표시용
  • 단, 구인자·대행소는 등급 확인 후 해당 구직자의 매칭을 취소할 수 있으며, 취소된 슬롯은 복원되어 다른 구직자가 신청 가능
대행소 등급 평가
  • 평가 주체: 구인자 또는 본사
  • 등급: 5단계 / 본사가 승급 또는 강등
  • 등급은 시스템에 영향을 주지 않으며 표시용

수정 규칙
  • 수정하려면 반드시 먼저 회수해야 함
  • 확정된 매칭이 1건 이상 존재하면 회수 불가 — 모두 취소된 상태여야 회수 가능
  • 회수 즉시 구직자 화면에서 비공개 처리, 신청 불가
  • 수정 완료 후 재상신 → 새로운 게시 상태로 공개
  • 임시저장 상태에서는 언제든 자유롭게 수정 가능
상태 전환 흐름
임시저장 → [상신] 상신(게시) → [회수, 확정0] 회수 → [재상신] 상신(게시)
상신(게시) → [인원 마감] 마감(자동) → [확정자 전원 취소] 상신(게시) 복원
기본 원칙
  • 구직자가 자율적으로 매칭 신청한다.
  • 신청 즉시 확정되며 슬롯이 차감된다. 슬롯이 없으면 신청 자체가 차단된다.
다중 매칭
  • 구인 글에 기재된 구인 인원수만큼 여러 구직자가 동시에 매칭 가능
  • 시스템은 구인 인원수 초과를 기술적으로 방지 (DB 트랜잭션 잠금 또는 원자적 카운터로 동시 신청 경합 처리)
  • 구인 인원 도달 시 해당 구인 글에 더 이상 신청 불가

  • 반경 거리 값은 구직자가 직접 설정
소개비 계산식
소개비 = 구인 글 일당 × 예정 기간 × 소개비율
소개비율: 구직자 소속 대행소가 자율 설정 (구직자는 가입 시 인지)
배분 구조
매칭 유형배분비고
동일 대행소 소개비 100% → 해당 대행소 보유 —
크로스 (다른 대행소) 소개비 50% → 구직자 대행소 보유
소개비 50% → 구인자 대행소 이체		 기준 요율: 구직자 소속 대행소 요율 적용
본사 수익
본사 납부액 = 대행소 실수취 소개비 × 30%
대행소 최종 수익 = 실수취 소개비 × 70%
본사는 전체 대행소 월매출 합산의 30% 수취
대금 흐름

  • 구직자는 구인자에게 일당을 직접 계좌이체로 수령
  • 구직자는 소개비를 소속 대행소에 별도 입금
